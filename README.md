# 🖥️ Calculadora Clementina

Una calculadora web inspirada en **Clementina**, la primera computadora científica utilizada en Argentina y un símbolo del desarrollo de la informática nacional.

Este proyecto fue desarrollado con **HTML, CSS y JavaScript** como ejemplo de una aplicación web estática y fue desplegado utilizando **Vercel**, integrando el repositorio alojado en GitHub.

## 🚀 Demo

https://calculadora-argentina.vercel.app/

## ✨ Características

- Operaciones matemáticas básicas
- Soporte para teclado
- Diseño retro inspirado en la informática de los años 60
- Mensajes históricos sobre Clementina
- Diseño responsive
- Sin dependencias externas

## 🛠️ Tecnologías

- Git

## ▶️ Ejecutar localmente

1. Clonar el repositorio

```bash
git clone https://github.com/TU_USUARIO/calculadora-argentina.git
```

2. Entrar al proyecto

```bash
cd calculadora-argentina
```

3. Abrir `index.html` en el navegador.

## 🌐 Despliegue

Este proyecto utiliza el flujo de despliegue continuo (CI/CD):

```
Visual Studio Code
        ↓
      Git
        ↓
     GitHub
        ↓
     Vercel
        ↓
Producción
```

Cada cambio enviado al repositorio mediante `git push` genera automáticamente un nuevo despliegue en Vercel.

## 📖 Sobre Clementina

"Clementina" fue el nombre con el que se conoció a la computadora Ferranti Mercury instalada en el Instituto de Cálculo de la Universidad de Buenos Aires en 1961. Su apodo surgió porque, al iniciarse, reproducía la melodía **"Oh My Darling, Clementine"**, convirtiéndose en uno de los íconos de la historia de la informática argentina.
